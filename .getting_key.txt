1. Navigate to Google AI Studio
• Open your browser and go to: https://aistudio.google.com/app/apikey
• (Note: You will need to log in with your Google account if you aren't already.)
2. Create a Project First (The Crucial Step)
• Do not click "Create API Key" yet. * Using your screen reader, navigate the page to find the section or button labeled "Projects" (often found in the navigation menu or dashboard area).  
• Activate the "Create Project" button.  
• Type a simple name (like "Accessible Bible") and press Enter to confirm. Wait a few seconds for the system to announce that the project has been created.
3. Generate the Key
• Now, navigate back to the main heading that says "Get API key" or "API Keys".  
• Use your Tab key to find the "Create API key" button and press Enter.  
• Because you already created a project, the interface should now allow you to simply select that project (or it may default to it) and generate the key without wrestling with the complicated "Create New" combo box.
4. Copy the Key
• A dialog will appear containing your new key (a long string of random letters and numbers).
• Use your Tab key to move past the text field containing the key.
• The very next tab stop is the copy button. It might read as "Copy API key to clipboard", an unlabeled graphic, or simply "Button". Press Enter on it to copy the key.

Return to Accessible Bible and use O to open the Options menu.
Navigate down to the option to add Gemini key and hit enter on this option.
You can use Control + V to paste your API key from your clipboard into this field in the Options menu.
You should be now able to use G and generate your own study plans

## 0.14.0 - Hybrid Relational Engine, Verse Menus, & Local Backups
### Added
* Breadcrumb Stack: Added a `navigationHistory` array. Teleporting pushes the current index; pressing `Backspace` pops the history to retrace steps.
* Anchor & Link System: 
    * `R` anchors a verse coordinate in memory.
    * `Alt + L` (Global) silently appends the anchored link to the current verse's note.
    * `Alt + L` (Local) injects the anchored link at the cursor in the Note Editor.
    * `Alt + G` (Global) scans the current note for `[[links]]` and auto-teleports (or opens a menu if multiple are found).
    * `Alt + G` (Local) teleports to the link currently under the text editor cursor.
* Vertical Axis Actions: 
    * `Up Arrow` instantly reads the current verse's note from the database (if it exists).
    * `Down Arrow` opens the temporary Verse Action Menu (`isMenuMode`).
* Verse Action Menu: Includes Options to Edit Note, Delete Note, and Copy Verse to system clipboard.
* Global Data Backups: `Shift + E` exports the `NOTES_STORE` as a JSON file. `Shift + I` imports and merges a JSON backup into the database via a stealth file input.

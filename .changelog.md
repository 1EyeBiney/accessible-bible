
## [0.6.0] - 2026-04-07
### Added
- Status Mission ('S'): Implemented a chapter-level statistics readout for rapid study orientation.
- Dynamic Analysis: Key 'S' now calculates and announces the total verse count and word count for the current chapter in real-time.
- Mode Safety: Added logic to ensure entering one search mode (B, C, or V) automatically clears any pending buffers or flags from other modes.
### Changed
- gemini.md: Updated Section 3 (Keybindings) to reflect the Digit Buffer mechanics and Section 4 to include the Status Mission requirements.
## [0.5.0] - 2026-04-07
### Added
- Digit Buffer System: Replaced `window.prompt` for 'C' and 'V' keys with a non-blocking numeric input mode.
- Real-time Audio Feedback: The engine now announces digits as they are typed for blind-user verification before jumping.
- Input State Management: Added `inputBuffer`, `isChapterMode`, and `isVerseMode` to the global state.
### Changed
- Refined Keyboard Interceptor: Consolidated numeric entry logic to prevent standard browser shortcuts from firing during coordinate entry.
## [0.4.0] - 2026-04-07
### Added
- First-Letter Book Search: Replaced the `prompt()` for the 'B' key with a chronological first-letter cycling system.
- Page Navigation: Implemented PageUp/PageDown for instant Chapter jumps and Shift + PageUp/PageDown for Book jumps.
- Search State: Introduced `isBookSearchMode` to intercept alphabetical keys for rapid indexing.
### Changed
- Improved Page Key Handling: Intercepted default browser scrolling to ensure focus remains on the text during jumps.
#E# [0.3.0] - 2026-04-07
### Added
- Canonical Sorting: Added in-memory sort to loadToMemory() to ensure book/chapter/verse order regardless of IndexedDB alpha-sorting.
- Coordinate Navigation: Implemented 'B' (Book), 'C' (Chapter), and 'V' (Verse) keys using accessible prompts for rapid jumping.
- Jump Logic: Added jumpTo() helper to sync currentVerseIndex with user-requested coordinates.

- Resolved "Alphabetical Bug" where 1 Chronicles preceded Genesis in the RAM cache.

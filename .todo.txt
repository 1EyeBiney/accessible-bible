### v0.47.0 — Remote Curriculum Gallery
- **Remote Fetch Engine:** Implemented `fetchRemoteCommentary()` to allow the engine to download and install JSON study plans directly from a URL, bypassing the need for local file dialogs.
- **Curriculum Gallery UI:** Added a "Browse Study Plans" submenu to the Options (`O`) menu. This features a pre-filled list of validated study plan URLs, allowing non-technical users to "point and click" to install new curriculum.
- **Auditory Confirmation:** Integrated TTS status updates to confirm successful remote installations or report connection errors.
Why this is more "Bulletproof":
1. 
No Files: The user never has to see a Windows/Mac file explorer.
2. 
No Typing: Since the URLs are pre-filled in an array, they don't even have to copy and paste. They just listen and pick.
3. 
Always Updated: If you update the Evangelism Explosion file on your server, the user gets the latest version the next time they "Pick" it from the list.
